### Question 12

Show the output produced by each of the following program fragments. Assume that i and j are int variables.

(a) i = 5;  
j = ++i * 3 - 2;  
printf("%d %d", i, j);

(b) i = 5;  
j = 3 - 2 * i++;  
printf("%d %d", i, j);

(c) i = 7;  
j = 3 * i-- + 2;  
printf("%d %d", i, j);

(d) i = 7;  
j = 3 + --i * 2;  
printf("%d %d", i, j);

### Solution

(a)  
j = ++i * 3 - 2 = (6 * 3) - 2 = 18 - 2 = 16  
i = 6

(b)  
j = 3 - 2 * i++ = 3 - (2 * 5) = 3 - 10 = -7  
i = 6

(c)  
j = 3 * i-- + 2 = (3 * 7) + 2 = 21 + 2 = 23  
i = 6

(d)  
j = 3 + --i * 2 = 3 + (6 * 2) = 3 + 12 = 15  
i = 6
